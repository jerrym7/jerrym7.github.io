My Projects

Fodongos

After covid everyone in the world got hit economically, specially stylists. People needed a safe way to get haircuts and wanted to know if stylists were taking precaution. Fodongos helped clear those doubts, how? Booking appointments online, knowing if the stylist were taking precautions (face mask, social distancing, and hand sanitizer), and maintaining the stylist open in a safer manner. Application is created in react native, for IOS and Android, with future launch of a website version in reactjs. Backend was created using ExpressJs and MongoDB for database.

Dictionary Chido

Languages are evolving with people using different words and some words being the same, but having different meanings. Dictionary Chido is there to help people around the globe. Everyone can add words to the online dictionary with a vote feature, to help people find the most popular meaning. This application was deployed to the public within a month it had over 1000+ downloads in Google Play and a 5 star rating. The web application has 1000+ visits in a month. Due to cost being expensive it was brought down. This application was created in react js and Android Studio (not for IOS) with backend in PHP, and MySQL as database. Rent My Driveway

This android application was created with the purpose to reduce the parking price from events. Ex. Super Bowl, if you live close by the event and you have extra space you can rent it to anyone that is looking for a parking spot, everything done in real-time. This was created on Android, backend was done in java and firebase as database.

Alpha Lion

This is an android game created to exercise the brain, creates random math problems which gets harder everytime you get the answer right. This game was created in a team of four for software engineering course following a set of guidelines with specifications. We took advantage of storage to save the best score and threads to limit the user with a time limit to answer each mathematical problem (10 seconds).
